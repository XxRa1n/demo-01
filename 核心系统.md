# 核心系统
## 1.武器系统
根据行动原语分类：
### 行为原语 → 武器 → 基类 对照

| 行为原语 | 命中方式 | 对应武器 | 工程基类 |
|---|---|---|---|
| `Seek` | 飞行单体 | 弩箭，匕首，回旋镖，火枪，冰锥术，竖琴 | ProjectileBase |
| `Spread` | 扇形多发 | 弓箭，宝石法术 | ProjectileBase |
| `Explosive` | 飞行体命中 / 到点爆炸 | 火球术 | ExplosiveProjectile |
| `Lob` | 抛物投掷、延时引爆 | 圣水瓶，矮人榴弹炮 | ExplosiveProjectile |
| `Beam` | 持续线性穿透、按帧 DPS | 激光 | Beam |
| `Pulse` | 扇形or环形扩散可设定弧度、逐层命中 | 声波，喷火 | Shockwave |
| `Orbit` | 环绕玩家轨道、持续 | 旋转飞斧，环绕圣经 | OrbitEntity |
| `Slash` | 瞬时扇形区域 + 击退 | 长剑，巨剑，棍子，长戟 | SlashArea |
| `Aura` | 持续圆形生效领域 | 竭心光环 | Zone |
| `Chain` | 命中后链式跳跃 | 闪电链 | Chain |
| `Turret` | 定点 / 环绕自主射击 | 机器人炮台 | Minion |
| `Follower` | 跟随自主攻击 | 猫，龙，地鼠，fairy，赛车 | Minion |
